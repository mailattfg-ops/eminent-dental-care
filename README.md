# Eminent Dental Care Website

A premium, high-end clinical website for Eminent Dental Care, built with Next.js, Tailwind CSS, and Framer Motion. This project showcases a modern, architectural approach to dental clinic web presence.

## 🚀 Features

- **Premium UI/UX**: High-fidelity clinical aesthetics with smooth, sophisticated animations.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop devices.
- **WhatsApp Integration**: Floating contact button and CTAs that redirect directly to clinical support.
- **Bento-Grid Facilities**: A modern showcase of state-of-the-art dental facilities.
- **Comprehensive Services**: Detailed sections for General, Cosmetic, and Pediatric dentistry.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
